# Action Planning Prompt

Based on the opportunity mapping output, recommend the next action.

Return:

1. Recommended action
2. Communication angle
3. Suggested CTA
4. Optional message draft if appropriate
5. Internal note
6. Risk warning
7. Human review requirement

Rules:

- Keep action low-friction when evidence is weak.
- Avoid pressure or unsupported claims.
- Do not push a product, service, or commercial offering unless it is explicitly provided in input.
