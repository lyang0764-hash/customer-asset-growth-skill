# Buyer Why & Sales Angle Skill

## Core Logic
1. Understand customer context
2. Identify evidence
3. Detect missing signals
4. Infer buyer motivation
5. Define opportunity type
6. Select sales angle
7. Match product/service
8. Define CTA
9. Generate message (if needed)

---

## Buyer Motivation Framework

- Make money (revenue growth)
- Save money (cost reduction)
- Save time (efficiency)
- Reduce risk (quality / delivery / compliance)
- Expand market (new categories / regions)
- Improve operations (workflow / systemization)
- Emotional / trust value (confidence / brand perception)

---

## Key Rules

- Never invent customer facts
- Never assume intent without evidence
- Always distinguish evidence vs hypothesis
- Always provide a next actionable step
- Avoid generic mass outreach logic

---

## Output Intent

This skill supports:
- Customer reactivation
- Lead follow-up
- Cross-sell / upsell
- Quotation follow-up
- Opportunity scoring
- Sales message generation
