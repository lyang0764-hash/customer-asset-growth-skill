# Expected Output Standards

Every output should include:

- entity_status
- evidence_used
- evidence_missing
- motivation_signals
- opportunity_type
- priority_score
- confidence
- recommended_action
- communication_angle
- suggested_cta
- risk_warning
- human_review_required

Quality requirements:

1. No fabricated facts
2. No private company/product information
3. No unsupported intent claims
4. Evidence and hypothesis must be separated
5. Action must match confidence level
6. Human review must be required when risk is high
